# -Wateen-
مشروع "وتين" (WETEEN): منصة التمريض المنزلي الذكية
"وتين" هي منصة ويب مركزية (Centralized Web Platform) تهدف إلى هيكلة وتنظيم سوق التمريض المنزلي في مصر. يعمل المشروع كحلقة وصل رقمية موثوقة تجمع بين المرضى الباحثين عن رعاية صحية آمنة، وبين الممرضين المؤهلين. يتبنى المشروع نموذج "دليل الخدمات والحجز" (Directory & Booking System) —المشابه لـ Booking.com— مبتعداً عن تعقيدات التتبع اللحظي (On-Demand Dispatch)، ليركز بدلاً من ذلك على الموثوقية، الخصوصية، وسهولة الوصول.

المشكلة والحل: يعالج "وتين" تحديات السوق المصري المتمثلة في غياب الموثوقية، تفاوت الأسعار، وصعوبة العثور على ممرض متخصص في الحيز الجغرافي القريب. يقدم الحل عبر منصة تتيح للمرضى تصفح ملفات الممرضين بشفافية وحجزهم بناءً على الكفاءة والموقع.

أبرز المميزات التقنية والذكية:

جواز السفر الطبي (Medical Passport): سجل رقمي مشفر لتاريخ المريض الطبي وحساسياته، لا يتم كشفه للممرض إلا بعد قبول الحجز لضمان أعلى درجات الخصوصية.

التقييم الذكي (AI Smart Rating): خوارزمية برمجية تمنح الممرضين الجدد "تقييماً أولياً" عادلاً بناءً على مؤهلاتهم وسنوات خبرتهم، مما يساعدهم على المنافسة فور الانضمام.

التحقق من الهوية (Identity Verification): نظام أمان يستخدم تقنيات الرؤية الحاسوبية (Computer Vision) لمطابقة صورة بطاقة الرقم القومي مع صورة "السيلفي" للممرض لمنع انتحال الشخصية.

البحث الموجه: فلترة دقيقة حسب المنطقة (Zone) والتخصص (Specialty).

البنية التقنية (Tech Stack): تم بناء "وتين" كـ (Educational MVP) باستخدام تقنيات قياسية ومستقرة:

Backend: إطار عمل Python Flask مع قاعدة بيانات SQLite.

Frontend: واجهات HTML5/CSS3/Vanilla JS متجاوبة وتدعم اللغة العربية (RTL).

يمثل "وتين" حلاً تقنياً متوازناً يجمع بين البساطة الهندسية والذكاء البرمجي لخدمة قطاع الرعاية الصحية في مصر بكفاءة وأمان.
